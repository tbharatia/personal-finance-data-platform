# Personal Finance Data Platform & Analytics

## Overview
This project builds an end-to-end data platform that ingests banking transaction data using Python, transforms it into a structured data warehouse, and delivers budgeting, spending analysis, and financial planning insights through Power BI dashboards.

## Data Pipeline
- Automated data ingestion from banking data sources using Python
- Data cleaning and validation workflows
- SQL-based transformation into warehouse tables
- Analytics-ready datasets for reporting

## Tools Used
- Python
- SQL
- Power BI
